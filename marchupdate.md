I wrote a TL;DR for the changes in the last CR and TR. This is heavily inspired by Grenton's video, which contains way more details and explanations on each topic : https://www.youtube.com/watch?v=3Ex2D2EKVks

It's also a focus for judges and players of skirmishes and other tournaments before Unleashed release, so I did not include set 3 exclusive rulings.

### CR Update
- HOT FEPR system : FEPR means fully resolving the chain if there is something in it (including priority passes). HOT is an uninterruptible task.
- Combat resolution is now
   - Combat damage
   - Combat cleanup (with heals and attackers recalls)
   - Resolve FEPR
   - Combat result (win/lose/no result)
   - Resolve FEPR
   - Establish control / conquer
   - Resolve FEPR
   - Combat end
- *Implications* : Assault now works with sunken temple. Imperial Decree will prevent draven from winning combat

- May abilities : Abilities including "you may" are now optional triggers, the player will chose to put the ability on the chain. If they does, they must resolve it.
   - Costs must be payed when the ability goes on the chain (for abilities in the format [Do X] to [Do Y], or Cost Within Instruction triggers)
   - Conditions must be met to be put on the chain.
- *Implications* : Monastery of Hirana needs a buff to be there before conquer to be able to go on the chain. Overzealous fan will kill itself to put its ability on the chain. Sona's ability will only go on the chain if she's at a battlefield.

- Battlefield control is not lost if there is either : 
   - A showdown
   - A combat
   - Any items on the chain
- *Implications* : Any "remove then play units" effects will be able to be played at a battlefield where the units are removed (cruel patron, arcane shift, deathknell...)

- An non-combat showdown will become a combat showdown if there is units from different players at that battlefield. (It does not wait for the non-combat showdown resolution, and can not go back to being an non-combat showdown)
- Attaching a card to the card it's already attached will do nothing. (Jax will not trigger brutaliser / aphelios by attaching the equipment to the same unit)
- New mechanics : Hunt / XP / Level / Ambush : set 3 only
- Legion defined : A different card must be played, but being finalized is enough. (Sun disk won't work on itself, but a countered spell will enable legion)
- Attached cards keep granted keywords (Giving temporary with Fading Memories will work as expected)
- Discounts for additional costs are applied immediately and only to that specific cost. (Can't play Bellows Breath for 0 energy with Vex + Ez)
### TR Update
- Clearer rewinds
- New warning system
- New facedown card error fix/penalty : If no card have been added to the zone the card come from (usually hand), it can be send back to hand and a random hidden card put in place instead.
- Hypothetical questions : always ask questions about the rules, not about the board
- Runes must be physically put at the bottom of rune deck when recycled. No more facedown runes to track pooled energy.
